# axiom Hackathon Deck

Files:
- `index.html`: 7-slide shortlist deck for the hackathon submission
- `styles.css`: deck styling and print layout

Export:
1. Open `index.html` in a Chromium browser.
2. Print to PDF with backgrounds enabled.
3. Use landscape `16:9` output with margins disabled.

Notes:
- The deck uses the new `axiom` brand.
- The current implementation in the repo is still under the previous package naming while the rename is in progress.
